  • button's with aria-label do not read aloud the button text nodes, but instead read the aria-label along with the button's role
  • paragraph's with aria-label first read the aria-label, then mention it's a group, and after a subsequent keypress read aloud the paragraph text
  • heading's with aria-label read the heading level then the aria-label text in place of the heading text
  • span's with aria-label first read the aria-label text and then announce the span as a group, subsequent keypress reads aloud the normal span text
  • article's with aria-label first read the aria-label text and then announces it as an article, subsequent keypress reads aloud the normal article text
  • aside's with aria-label first read the aria-label text and then announces it as a complimentary, subsequent keypress reads aloud the normal aside text
  • section's with aria-label first read the aria-label text and then announces it as a region, subsequent keypress reads aloud the normal section text
  • nav's with aria-label first read the aria-label text and then announces it as a navigation, subsequent keypress reads aloud the normal section text
  • header's with aria-label first read the aria-label text and then announces it as a group, subsequent keypress reads aloud the normal header text
  • footer's with aria-label first read the aria-label text and then announces it as a group, subsequent keypress reads aloud the normal footer text
